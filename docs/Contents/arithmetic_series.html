Arithmetic Series#

Derivation#

Let us start by adding the integers from 1 to 5. The total is,

\[1 + 2 + 3 + 4 + 5 = 15\]

Rewrite the sum as,

\[(5 - 0) + (5 - 1) + (5 - 2) + (5 - 3) + (5 - 4) = 15.\]

Notice that, \(5 - 1\) is cancelled out by the \(-4\) in \((5 - 4)\). \((5 - 2)\) is cancelled out by \(-3\) in \((5 - 3)\). Therefore, the equation above can be simplified to,

\[5 + 5 + 5 = 15.\]

Now let us do the sum from 1 to 6.

\[(6 - 0) + (6 - 1) + (6 - 2) + (6 - 3) + (6 - 4) + (6 - 5).\]

This reduces down to,

\[(6 - 3) + 6 + 6 + 6.\]

The situation looks slightly different when the maximum number is even. Let us come up with a general rule for the sum when \(n\) is odd. Consider the follow sum,

\[(n - 0) + (n - 1) + (n - 2) + \dots + (n - (n - 1)).\]

Since \(n\) is odd, not including \((n - 0)\), there are an even number of remaining terms since \(n - 1\) must be even.

Notice that \((n - 1)\) is cancelled out by \(- (n - 1)\) from \((n - (n - 1))\). In fact all terms \(n - i\) where \(i \leq (n - 1) / 2\) will be cancelled out by its corresponding term in \(n - (n - i)\) for \(i \gt (n - 1) / 2\). Let us prove this for the case where \(i = (n - 1) / 2\), which is the midpoint. In this case on the left side, we have,

\[n - \frac{n - 1}{2} = \frac{n + 1}{2}.\]

Its corresponding term on the right will be \(i = (n - 1) / 2 + 1\). Resulting in,

\[n - (\frac{n - 1}{2} + 1) = n - (\frac{n + 1}{2})\]

The terms therefore cancel. Cancelling out all these terms leaves us with,

\[n + \frac{n - 1}{2}n = \frac{n^2 - n + 2n}{n} = \frac{n(n + 1)}{2}\]

Now let us consider the cases were \(n\) is even. Again we have the same sum,

\[(n - 0) + (n - 1) + (n - 2) + \dots + (n - (n - 1)).\]

But if we ignore the first term there are an odd number of remaining terms meaning there will be one term left when we cancel everything out. The term that is left over will be,

\[n - \left( \frac{n - 2}{2} + 1 \right).\]

The fraction here is floored midpoint. Cancelling everything out again means that in the end we have,

\[\begin{split}\begin{align} & n + \left(n - \left( \frac{n - 2}{2} + 1 \right) \right) + \frac{n - 2}{2} n \\ = & n + \frac{n}{2} + \frac{n^2 - 2n}{2} \\ = & \frac{2n + n + n^2 - 2n}{2} \\ = & \frac{n(n + 1)}{2} \end{align}\end{split}\]

So, it does not matter \(n\) is even or odd, we still get the same result.

In summary, the formula is,

\[\sum_{k = 1}^n k = \frac{n(n + 1)}{2}\]

Simpler Proof#

The proof above is more complicated than it needs to be and in fact there is a much simpler proof. Consider the following sum.

\[S_n = 1 + 2 + \dots + n.\]

Consider the same sum in reverse order.

\[S_n = n + (n - 1) + \dots + 1.\]

Add the two together.

\[\begin{split}\begin{align} S_n + S_n & = n + ((n - 1) + 1) + ((n - 2) + 2) + \dots (1 + (n - 1)) + n \\ & = n + n + n + \dots n + n \\ & = n(n + 1) \\ \end{align}\end{split}\]

Therefore,

\[\begin{split}\begin{align} 2 S_n & = n(n + 1) \\ S_n & = \frac{n(n + 1)}{2} \\ \end{align}\end{split}\]

And finally,

\[\begin{equation} \sum_{k = 1}^n = \frac{n(n + 1)}{2} \end{equation}\]
